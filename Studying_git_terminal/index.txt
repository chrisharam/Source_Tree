git init .
: 해당 파일(.)을 초기화(init)

ls -al
: 해당 파일에 있는 모든 것을 출력

nano 파일명.txt
: 파일 만들기

control + x
: nano 나가기

y/n
: 저장 유무

enter
: 나가기

cat 파일명.txt
: 뒤에 따라오는 파일의 내용을 출력

git add 파일
--> git이 해당 파일을 추적하기 시작함.

git log
: git의 버전 상태 역사를 모두 봄.

기본적인 나가기
: q

git log --stat
: 각 버전의 세부 변경사항을 모두 봄 

git diff
: 이미 커밋된 것에서, 달라진 점을 보여줌(새로운 수정사항)

git reset --hard
: 현재 버전을 삭제하고, 이전의 버전으로 돌아감

git log -p
: 각각의 버전의 패치 버전(버전에 따라 변경된 점을 보여줌)
